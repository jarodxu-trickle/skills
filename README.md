# Claude Code Skills

A collection of reusable Claude Code skills for common tasks.

## Skills

### fix-thinking-signature

Repair corrupted Claude Code session files that fail with "Invalid signature in thinking block" errors. This happens when switching between different AI models (e.g., Claude Opus, Sonnet, Haiku, or third-party models) mid-session.

**Install:**

```
claude install-skill /path/to/fix-thinking-signature
```

Or copy the `fix-thinking-signature/` directory to `~/.claude/skills/`.
